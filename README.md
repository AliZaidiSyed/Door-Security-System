# Door Security System

A microcontroller-based door security project that uses a keypad and LCD to control access with a password/PIN system.

## Features
- PIN/password verification via keypad
- LCD status messages (Access Granted/Denied)
- Basic security logic with retry/lock behaviour

## Hardware
- Arduino (or compatible board)
- 4x4 / 4x3 Keypad
- I2C LCD (16x2)
- Wires + power source

## How to Run
1. Open the `.ino` file in Arduino IDE
2. Install required libraries (Keypad, LiquidCrystal_I2C)
3. Select board & COM port
4. Upload and test on hardware
